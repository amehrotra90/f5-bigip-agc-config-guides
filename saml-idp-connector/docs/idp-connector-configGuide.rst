IdP Connector Configuration Guide
========================================================================

Usage
------

Follow the steps below to configure a use case with Access Guided Configurations.

#. Logon to the BIG-IP user interface and click :menuselection:`Access -> Guided Configuration`.
#. Select the :guilabel:`Federation` category.
#. For :guilabel:`SAML Service Provider`, select a template method for Identity Provider and refer to the IdP Connector configuration guide specific to the vendor in the table of contents on the left.

SaaS Application Configuration Guide
======================================================================================

Usage
------

Follow the steps below to configure a use case with Access Guided Configurations.

#. Logon to the BIG-IP user interface and click :menuselection:`Access -> Guided Configuration`.
#. Select the :guilabel:`Federation` category.
#. For :guilabel:`SAML Identity Provider for Applications`, select a template method for Saas Applications and refer to the SaaS Applications configuration guide specific to the application in the table of contents on the left.
